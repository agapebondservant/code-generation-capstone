## Servlet Compilation: CLASSPATH Entries

- · The servlet classes (usually in install\_dir/lib/servlet.jar)
- · The JSP classes (usually in install\_dir/lib/jsp.jar, ...jspengine.jar, or ...jasper.jar)
- · The top-level directory of servlet installation directory (e.g., install\_dir/webpages/WEB-INF/classes)

## Tomcat 3.0 Standard Directories

- · install\_dir/webpages/WEB-INF/classes Standard location for servlet classes.
- · install\_dir/classes

Alternate location for servlet classes.

- · install\_dir/lib

Location for JAR files containing classes.
